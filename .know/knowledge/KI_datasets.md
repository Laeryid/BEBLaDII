# KI: Datasets

## Что это
Описание источников данных, используемых для обучения и инференса в проекте BEBLaDII. Данные хранятся локально в формате Parquet для обеспечения совместимости и скорости.

## Перечень датасетов
| Название | Путь | Описание | Формат |
|---|---|---|---|
| **Magpie-Reasoning-V2** | `data/magpie_reasoning/` | 250k CoT примеров. | Parquet (12 shards) |
| **OpenThoughts-114k** | `data/open_thoughts/` | 114k высококачественных цепочек рассуждений. | Parquet (6 + 12 shards) |
| **CulturaX (Sample)** | `data/CulturaX/` | Огромный мультиязычный датасет (RU/CS части). | Parquet (2 files) |

## Неочевидные детали
- **CulturaX (RU/CS)**: В директории `data/CulturaX/` находятся первая часть русского корпуса (`ru_part_00000.parquet`) и чешского корпуса (`cs_part_00000.parquet`). Данные предназначены для префиксного обучения или индексации.
- **Прямое скачивание**: Файлы скачаны напрямую (через скрипт или вручную) в формате Parquet.
- **Структура**: Оба датасета имеют поддиректории `data/` и `metadata/` (специфика репозиториев на HF).
- **Отсутствие зависимостей**: Для скачивания не использовалась библиотека `datasets`, что минимизирует зависимости окружения.

## Типичные ошибки
- **Загрузка через pandas**: `pd.read_parquet("path/to/file.parquet")` работает для одиночных файлов. Для работы с набором файлов рекомендуется использовать специализированные библиотеки (будут добавлены позже).
